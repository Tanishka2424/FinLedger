# 💳 Banking Transaction System API

A backend banking transaction system built using Node.js, Express.js, and MongoDB.  
This project simulates core banking features like user authentication, account management, balance tracking, and secure money transfers.

## 🚀 Features

- User Registration & Login
- JWT Authentication
- Secure Password Hashing
- Create Bank Accounts
- Check Account Balance
- Deposit & Withdraw Money
- Transfer Money Between Accounts
- Transaction History
- RESTful APIs
- MongoDB Database Integration
- Error Handling & Validation

---

## 🛠️ Tech Stack

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

### Authentication
- JWT (JSON Web Token)
- bcryptjs

### Other Tools
- dotenv
- nodemon
- Postman
